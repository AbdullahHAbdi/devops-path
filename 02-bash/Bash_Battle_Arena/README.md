Bash Battle Arena (Levels 1–15) — Summary

Through completing Levels 1–15 of the Bash Battle Arena, I strengthened my Bash scripting skills and gained practical experience writing automation scripts for real-world system tasks. 

I practiced core scripting fundamentals including variables, conditionals, loops, and functions, and built progressively complex scripts across 15 levels. 

I learned how to handle user input and script arguments, parse files line by line, and manipulate text data using tools such as `grep`, `awk`, `sed`, and `wc`. I also worked with file system operations including searching with `find`, sorting with `sort`, and archiving with tar. 

The later challenges introduced practical DevOps-oriented scripting concepts including disk space monitoring with threshold alerts, automated directory backups with rotation, real-time directory change detection using MD5 checksums, and configuration file parsing. 

These exercises helped build a stronger foundation for writing maintainable Bash scripts and automating system administration tasks in real-world DevOps environments.
